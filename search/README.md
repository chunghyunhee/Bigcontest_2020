## 승률예측 방법?
- 설명을 보니 잔여시즌 승률 팀타율 방어율 계산함
- 잔여시즌이라면 후반기이므로 post-season 진출이 걸려있으므로 상위 5개팀은 지표가 좋을것
- 6~7위권 팀도 5위팀과 차이가 별로 없다면 꽤나 괜찮을수도
- 8~10위팀은 아마 성적이 나빠지지 않을까 예상함
